ADDING ELEMENTS TO ARRAY To add an element to the end of an array, you can use the push method. Add .push to the array name, followed by the element you want to add inside parentheses, like this: var animals = []; animals.push("Cat"); 1 animals.push("Dog"); 2 animals.push("Llama"); 3 animals; ["Cat", "Dog", "Llama"] animals.length; 3 Here we create an empty array with var animals = [];, and then use the push method to add "Cat" to the array. Then, we use push again to add on "Dog" and then "Llama". When we display animals;, we see that "Cat", "Dog", and "Llama" were added to the array, in the same order we entered them. The act of running a method in computer-speak is known as calling the method. When you call the push method, two things happen. First, the element in parentheses is added to the array. Second, the new length of the array is returned.
